# 坐标系选择分析：平面极坐标系与竖直笛卡尔坐标的组合

## 问题背景与坐标系需求

本文研究圆柱镜面反射的几何建模问题，涉及三个空间要素：纸面（平面）、圆柱镜面侧面（曲面）以及观察者（点）。需要建立一个能够高效描述光线反射路径、便于图案设计与数值计算的坐标系。候选方案包括全局直角坐标系、空间柱坐标系（空间极坐标系）以及平面极坐标与竖直坐标组合的混合体系。通过分析问题固有的几何对称性和映射结构，本文选择**平面极坐标系**描述纸面，辅以**竖直笛卡尔坐标** $z$ 描述高度，而镜面则自然地用**柱坐标** $(\theta,z)$ 表示。

## 直角坐标系与空间极坐标系的局限性

### 直角坐标系的冗余约束
若采用全局直角坐标系 $(x,y,z)$，纸面位于 $z=0$，圆柱侧面由方程 $x^2+y^2=R^2$ 约束。此时：

- 柱面上任意点的表示需同时满足二次约束，增加了参数化的复杂性。
- 镜面法向量 $\mathbf{n}=(\cos\theta,\sin\theta,0)$ 在直角坐标下需反复使用三角函数或根式表达，表达式冗长。
- 反射路径的推导会形成含 $\sqrt{x^2+y^2}$ 的非线性方程，不利于解析化简与数值计算。
- 纸面图案围绕圆柱中心呈放射状分布，直角坐标的矩形网格与图案的环形结构不匹配，导致边界判定、采样效率低。

### 空间极坐标系（柱坐标系）的不完全适用
空间柱坐标系 $(r,\theta,z)$ 能较好地描述圆柱侧面（$r\equiv R$）和纸面（$z=0$ 且 $r,\theta$ 自由）。但若将之作为全局坐标系，纸面上的点需用 $(r,\theta,0)$ 表示。这本质上是**平面极坐标**加 $z$ 坐标。因此，空间柱坐标系自然退化为“平面极坐标系 + 竖直坐标”的组合。由于纸面严格限制在 $z=0$ 平面，无需三维径向坐标的完全自由度，单独保留二维极坐标更为直接且与印刷、绘图坐标系衔接更紧凑。

## 平面极坐标系 + 竖直笛卡尔坐标的优势

针对本题，将纸面所在平面独立采用**平面极坐标系** $(\rho,\phi)$，竖直高度使用**常规笛卡尔坐标** $z$，镜面采用**柱坐标** $(R,\theta,z)$。这一混合坐标系具备以下关键优势：

### 1. 最大化利用圆柱的旋转对称性
镜反射问题中，圆柱侧面是轴对称的。其几何形状及法线方向天然适合用角度 $\theta$ 和高度 $z$ 描述。在极坐标框架下，镜面法向量简洁地表示为分量 $(\cos\theta,\sin\theta)$，反射公式中的各项可利用三角恒等式直接化简为 $\cos2\theta$、$\sin2\theta$ 等形式，得到极其紧凑的正向映射表达式。若使用直角坐标，这些对称性会被代数复杂度掩盖。

### 2. 反射映射的自然表达
推导得到的纸面像点极坐标 $(\rho,\phi)$ 与镜面坐标 $(\theta,z)$ 之间的关系为：

$$
\begin{aligned}
\rho &= \sqrt{\alpha^2 R^2 + \beta^2 L^2 + 2\alpha\beta R L\cos\theta},\\
\phi &= \operatorname{atan2}\big(\alpha R\sin\theta + \beta L\sin2\theta,\; \alpha R\cos\theta + \beta L\cos2\theta\big).
\end{aligned}
$$

该映射在极坐标下呈现出关于 $\theta=0$ 的对称性，且径向 $\rho$ 与高度 $z$（通过系数 $\alpha,\beta$）之间具有近乎解耦的趋势：$z$ 主要控制径向拉伸，$\theta$ 主要控制纸面方位角。这种特性利于理解和优化图案变形。

### 3. 纸面图案的辐射状分布
反射产生的纸面图案以圆柱底面中心为核心向外辐射，呈近似环形或扇形分布。采用极坐标表达纸面坐标，可以直观地判断图案的径向范围 $[\rho_{\min},\rho_{\max}]$ 和角度跨度，从而方便地进行A4纸张边界约束的判定与圆柱摆放位置的调整。极坐标的输出仅需一步坐标转换即可变为打印所需的直角坐标，不损失精度。

### 4. 竖直坐标 $z$ 保留笛卡尔坐标的原因
高度方向不具有旋转周期性和径向发散性，镜面图案在高度上通常为线性分布，观察者高度 $H_e$ 也是自然的高度值。若对 $z$ 也引入某种“极径”度量，会破坏线性结构并导致映射公式额外增加非仿射变换。保留 $z$ 为普通笛卡尔坐标，使得系数 $\alpha(z),\beta(z)$ 为 $z$ 的简单有理函数，并且便于设定圆柱高度 $H$、镜面图案的上下边界等参数。在数值计算中，$z$ 网格可以保持规则间隔，简化插值与积分。

## 结论

综合考虑几何对称性、映射形式简洁性、图案分布特征以及计算便利性，本文选择**以圆柱底面圆心为极点、纸面所在平面建立平面极坐标系**，同时**竖直高度 $z$ 沿用笛卡尔坐标**，镜面则对应地使用柱坐标 $(R,\theta,z)$。这一组合显著降低了理论推导与数值算法设计的难度，并为后续参数优化、变形分析与艺术加工提供了直观的数学框架。
