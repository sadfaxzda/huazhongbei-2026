# Q1 实验记录：角度放大效应适用边界验证

## 实验背景

正向映射中 $\phi \approx 2\theta$ 的线性近似（角度放大效应）在 $|\theta| \lesssim \pi/4$ 时成立。本记录验证该边界对实际渲染结果的影响。

---

## 实验一：蒙娜丽莎（90° 张角，预期成功）

| 参数 | 值 |
|------|-----|
| R | 3.5 cm |
| D | 25.0 cm |
| z_E | 35.0 cm |
| z 范围 | [0.2, 12.0] cm |
| theta_max | π/2（90°） |
| 采样网格 | m=600, n=400 |

**预期结果：** 纸面图案形成完整的约 180° 半环，人像比例正常，无自交叠。φ≈2θ 近似在整个张角范围内成立，插值散点分布均匀。

---

## 实验二：小猫（90° 张角，已修正，预期成功）

| 参数 | 值 |
|------|-----|
| R | 3.5 cm |
| D | 25.0 cm |
| z_E | 35.0 cm |
| z 范围 | [0.2, 9.0] cm |
| theta_max | π/2（90°，从 120° 修正） |
| 采样网格 | m=400, n=300 |

**预期结果：** 纸面图案形成约 180° 半环，卡通猫形态完整，无渲染错误。

---

## 对照实验：小猫（120° 张角，已观测失败）

| 参数 | 值 |
|------|-----|
| theta_max | 2π/3（120°） |
| 其余参数 | 同实验二 |

**实际现象：** 两翼（$|\theta| > 45°$ 区域）的投影点在纸面上出现自交叠，图案边缘渲染错乱，无法识别卡通轮廓。原因：$2\theta$ 进入第二象限后 $\cos 2\theta < 0$，被忽略的 $\alpha R$ 项不再是小量，$\phi$ 对 $\theta$ 不再单调，散点集在极角方向发生折叠。

---

## 结论

角度放大效应 $\phi \approx 2\theta$ 的有效适用范围为：

$$|\theta| \lesssim \frac{\pi}{4} \quad \Longleftrightarrow \quad \text{镜面总张角} \lesssim 90°$$

在此范围内，纸面图案近似为均匀展开的半环，渲染质量高。超出此范围需要引入更完整的映射模型或限制张角。

详细数学推导见 `docs/pixel grid.md` §3.6。
