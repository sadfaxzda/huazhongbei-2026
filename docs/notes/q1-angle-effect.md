# Q1 实验记录：角度放大效应近似边界验证

## 实验背景

正向映射中 $\phi \approx 2\theta$ 的线性近似（角度放大效应）在 $|\theta| \lesssim \pi/4$ 时精度较高。该边界只限制近似公式的解释范围，不等价于精确映射的可行边界。实际生成纸面图案时应使用完整公式

$$
\phi(\theta,z)=\operatorname{atan2}\!\bigl(\alpha R\sin\theta+\beta D\sin2\theta,\;\alpha R\cos\theta+\beta D\cos2\theta\bigr),
$$

并通过数值单调性和散点分布检查判断是否自交。

---

## 实验一：蒙娜丽莎（130° 张角，精确公式生成）

| 参数 | 值 |
|------|-----|
| R | 35 mm |
| D | 250 mm |
| z_E | 350 mm |
| z 范围 | [2, 120] mm |
| theta_max | 13π/18（130°） |
| 采样网格 | m=600, n=400 |

**结果说明：** 由于使用完整的 $\phi(\theta,z)$ 公式而非 $\phi\approx2\theta$ 近似，$130^\circ$ 张角下仍可生成连续纸面图案。此时纸面极角不再严格等于 $2\theta$，但数值映射仍可保持可用的单调展开关系。对蒙娜丽莎案例，$130^\circ$ 已接近 A4 下边界，继续增加会引入更明显的裁切。

---

## 实验二：小猫（130° 张角，精确公式生成）

| 参数 | 值 |
|------|-----|
| R | 35 mm |
| D | 250 mm |
| z_E | 350 mm |
| z 范围 | [2, 90] mm |
| theta_max | 13π/18（130°） |
| 采样网格 | m=400, n=300 |

**结果说明：** 与蒙娜丽莎案例一致，小猫案例采用完整映射公式生成。$130^\circ$ 超出 $\phi\approx2\theta$ 的线性近似边界，但不必然导致自交叠；由于小猫高度区间较短，其 A4 边界余量比蒙娜丽莎更大。

---

## 对照说明：90° 近似边界不是物理上限

| 参数 | 值 |
|------|-----|
| 近似公式 | $\phi\approx 2\theta$ |
| 高精度范围 | 总张角约 90° 以内 |
| 精确公式 | 可继续用于 120°、130° 等更大张角 |

过去将 $90^\circ$ 写成“有效上限”，实际指的是 $\phi\approx2\theta$ 这一近似关系的可靠范围。越过该范围后，不能再用 $2\theta$ 判断纸面包角和单调性；应回到精确公式计算 $\phi(\theta,z)$。在当前参数 $R=35\text{ mm},D=250\text{ mm},z_E=350\text{ mm}$ 下，$130^\circ$ 张角仍可由精确公式生成，不应被判为必然失败。

---

## 结论

角度放大效应 $\phi \approx 2\theta$ 的高精度适用范围约为：

$$|\theta| \lesssim \frac{\pi}{4} \quad \Longleftrightarrow \quad \text{镜面总张角} \lesssim 90°$$

在此范围内，纸面图案可近似理解为均匀展开的半环。超出此范围时，近似解释失效，但完整映射模型仍可使用；最终是否可行取决于精确公式下的散点分布、插值质量和 A4 边界容纳情况。
