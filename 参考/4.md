# 第一篇拆解：基于历史数据的晶硅片产销策略优化

## 1. 问题抽象

| 假设 | 内容 |
|------|------|
| 产销关系 | 0 库存，产销平衡 |
| 利润口径 | 税前利润；净利润 = 0.85 × 税前 |
| 外部因素 | 前三问不考虑政策/市场，仅用企业内部数据 |
| 成本简化 | 1–8 月价格不变的耗材视为定值 |
| 人工简化 | 固定部分合并为常数，浮动部分（0.04 元/片）归因到销量 |

核心思路：用弹性系数把 40+ 个变量筛到 Top 50%（10 个左右），只留四型硅片销量/售价、单晶方棒单价/单耗、水价/水耗、电价/电耗。

## 2. 预测模型

| 模型 | 用在哪 | 理由 | 平均误差 |
|------|--------|------|----------|
| ARIMA(0,1,0) | 弃用 | R² = -0.814，拟合曲线滞后于原序列 | — |
| 灰色 GM(1,1) | 售价、方棒单价、理论单耗 | 趋势性下降+小样本 | 0.11%–7.25% |
| Holt-Winters | 销量、水电耗量/费用 | 波动性强，水平+趋势双分量更适配 | 0%–16.28% |

按因子特征分配模型，不一刀切。

## 3. 优化模型

| 项 | 值 |
|----|-----|
| 算法 | 改进粒子群 PSO |
| 决策变量 | 20 维 |
| 约束 | 问题二给出的 9 月变化区间 |
| 超参数 | c₁=0.2, c₂=0.2, w=0.6（网格搜索） |
| 种群/迭代 | 200 / 500 |
| 结果 | 9 月最大利润 12,546,425.61 元 |

选 PSO 的理由：目标函数含 Q×P、Q×G×S 这类乘积项（非线性、连续），20 维带区间约束，PSO 无需梯度且收敛稳定。

## 4. LLM 融合

| 步骤 | 做法 |
|------|------|
| 爬虫源 | chinapv.org.cn、miit.gov.cn、gov.cn、chinaidr.com 等 |
| 关键词 | "晶硅"、"硅片" |
| 预处理 | 分句，每句 ≤ 50 字符（推测：为 BERT 输入长度控制） |
| 标注 | 对 5 类变量（销量/售价/方棒价/单耗/水电费）分别打分 ∈ [-1, 1] |
| 模型 | bert-base-chinese 微调（二分类头） |
| 量化公式 | P_t = P₀·(1 + ∂·S_t)，∂=0.2（引文献 [7] 取 0.1–0.3 居中） |
| 耦合方式 | 算 5 个变量的均值系数 α/β/γ/δ/ε，更新 PSO 上下界后重新求解 |

结果 11,733,734.73 元 < 纯历史数据版本，作者解释为市场/政策负面信号压制销量。

## 5. 验证手段

| 手段 | 具体做什么 |
|------|----------|
| 弹性系数排序 | 逐因子做 +1% 扰动，按 \|E\| 取 Top 50% |
| 留一月验证 | 1–7 月拟合 8 月，算平均相对误差选模型 |
| 模型 vs 实际 | 8 个月税前利润逐月对比（8 月差 ≈ 4.7%） |
| 残差正态性 | Shapiro-Wilk 检验；不正态时用 max\|e\| 替代 σ 做区间 |
| 横向淘汰 | ARIMA 因 R² 为负明确剔除，而非三个都留 |

## 6. 可复用套路

| # | 套路 | 适用场景 |
|---|------|---------|
| 1 | 弹性系数 E = (dπ/dx)·(x/π)，按 \|E\| 降序取 Top 50% | 任何多变量利润模型的特征筛选 |
| 2 | 按因子形态分配预测模型：趋势型→灰色，波动型→Holt-Winters；用 ARIMA 做陪跑对照 | n ≤ 12 的月度时序 |
| 3 | PSO + 网格搜索 (c₁, c₂, w)，决策变量直接取预测区间作边界 | 10+ 维非线性带区间约束优化 |
| 4 | 情感得分量化公式 P_t = P₀·(1 + ∂·S_t)，∂ 取 0.1–0.3 | LLM 文本输出耦合回数值模型 |
| 5 | 用 1–7 月预测 8 月做模型筛选 → 再用 1–8 月全量预测 9 月 | 小样本时序的"伪验证集" |
