# 华中杯 B 题"反射的艺术"作战计划

## 一、技术栈

| 环节 | 工具 | 说明 |
|---|---|---|
| 主语言 | Python 3.10+ | Python 生态更配合AI  |
| 数值计算 | numpy, scipy | 反射几何、优化求解 |
| 图像处理 | opencv-python, Pillow, scikit-image | 核心依赖，边缘检测、二值化、形态学 |
| 插值重采样 | scipy.ndimage.map_coordinates | Q1 反向光追的关键工具 |
| 频域分析 | numpy.fft | Q2/Q3 兼容性分析用 |
| 可视化 | matplotlib, PIL | 最终出图 |
| 3D 渲染 | Blender | 做一张高质量真实感反射图贴论文十分加分 |
| 版本管理 | GitHub 私有 repo | 必须用，否则覆盖代码很容易出问题 |
| 文档协作 | 飞书文档 | 实时多人编辑，比 Git 写 LaTeX 快 |
| 论文排版 | LaTeX| 最后半天迁移进来 |

**B 题相比 A/建模题的特殊依赖**：
- 核心是 `scipy.ndimage.map_coordinates`（反向光追后的像素采样）
- 不需要任何预测算法（没有时间序列数据）
- 不需要 LLM API（不存在 Q4 LLM 融合）

- 白板推导反射映射 T 的完整公式：
  - 镜面参数化：(θ, z) → P = (x₀ + R cosθ, y₀ + R sinθ, z)

### Day 3（48-72h）：Q3 兼容性理论 + 论文封稿

#### 第 8 阶段（48-58h）：Q3 兼容性判据 + 构造性算法

**A**（核心理论）：

列出 5 条兼容性条件，每条给可计算的数值判据：

1. **拓扑兼容**：F 和 T⁻¹(G) 的连通分量数差 ≤ ε₁
2. **尺度兼容**：镜面每行对应的纸面环形面积比 → 细节密度上界公式
3. **几何对称兼容**：F 极坐标 FFT 主频 vs G 直角坐标 FFT 主频，相关系数 ≥ ε₂
4. **可视区域兼容**：G 的支撑集必须落在纸面可视环内
5. **艺术容忍度**：差异图 D = G − T(F) 的高误差区域占比 ≤ ε₃，且高误差区域纹理复杂度 ≥ ε₄（能用装饰掩盖）

**B**（算法实现）：

实现构造性算法：

```
输入：纸面候选 F、镜面候选 G
Step 1: 检查 5 条兼容性判据，全部通过 → Step 2
Step 2: 计算差异图 D = G − T(F)
Step 3: 在 D 高误差区域生成装饰线条掩盖
Step 4: 输出艺术加工后的 F' 和渲染后的 G'
判决：可行性得分 ∈ [0, 1]
```

跑 2 组案例：
- 案例 1（可行）：F = 向日葵，G = 太阳光线 → 判据通过、出图
- 案例 2（不可行）：F = 蒙娜丽莎，G = 小猫 → 哪条判据失败、定量反例

**C**（论文 + 图）：
- 写 Q3 章节
- 画算法流程图
- 把两个案例的判据得分表格化

**产出**：Q3 章节 + 算法伪代码 + 2 个案例（1 正 1 反）
